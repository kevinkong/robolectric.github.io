Setup

These days it is simple to set up a bare-bones Android project that uses Maven and Robolectric. Just use Deckard.

Deploying .apks

Deckard uses the maven-android-plugin which makes it easy to add Android to your Maven project.

Importing into an IDE

If you’re using JetBrains' excellent IntelliJ IDEA, just open your pom.xml as a project, and you’re set.

If you’re using Eclipse, you can try to get it to work by following the eclipse quick start instructions.

When running tests from within IntelliJ or Eclipse, be sure to run tests using the JUnit runner and not the Android Test runner.
